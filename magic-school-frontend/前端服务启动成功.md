## 🔒 防火墙配置

如果无法从外部访问，请确保以下端口已在防火墙中开放：

### 需要开放的端口

| 端口 | 协议 | 说明 |
|------|------|------|
| 3000 | TCP | API 后端服务 |
| 5000 | TCP | 智能体后端服务 |
| 5173 | TCP | 前端开发服务器 |
| 8765 | TCP | WebSocket 服务 |

### 阿里云 ECS 安全组配置

1. 登录阿里云控制台
2. 进入「云服务器 ECS」->「实例」
3. 找到实例 ->「安全组」->「配置规则」
4. 添加入方向规则：

```
端口范围: 3000/3000, 5000/5000, 5173/5173, 8765/8765
授权对象: 0.0.0.0/0
协议: TCP
```

## 🐛 常见问题

### 问题 1: 无法访问前端页面

**原因**: 防火墙未开放 5173 端口

**解决方案**:
1. 检查服务器防火墙：`sudo ufw status`
2. 开放端口：`sudo ufw allow 5173`
3. 配置阿里云安全组（见上文）

### 问题 2: 页面空白

**解决方案**:
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签的错误信息
3. 查看 Network 标签的请求状态
4. 参考页面空白问题排查指南

### 问题 3: API 请求失败

**检查项**:
1. 后端服务是否正常运行
2. 前端环境变量配置是否正确
3. 防火墙是否开放 3000 端口
