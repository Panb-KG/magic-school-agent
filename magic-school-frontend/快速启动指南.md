# 魔法课桌学习助手 - 前端快速启动指南

## 🚀 一键启动（推荐）

### Linux / macOS

```bash
./scripts/start_frontend_dev.sh
```

### Windows

```bash
scripts\start_frontend_dev.bat
```

脚本会自动：
1. ✅ 检查 Node.js 和 Python 环境
2. ✅ 安装所需依赖
3. ✅ 启动 Mock API 服务器（端口 3000）
4. ✅ 启动前端开发服务器（端口 5173）

---

## 📖 手动启动

如果自动脚本无法运行，可以手动启动：

### 步骤 1: 启动 Mock API 服务器

```bash
# 安装 Python 依赖
pip install fastapi uvicorn pydantic python-jose[cryptography]

# 启动 Mock API 服务器
python scripts/mock_api_server.py
```

### 步骤 2: 启动前端开发服务器

打开新的终端窗口：

```bash
cd magic-school-frontend

# 安装前端依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

### 步骤 3: 访问应用

打开浏览器访问：**http://localhost:5173**

## 🔧 常见问题

### Q: 脚本无法执行？

**Linux / macOS**:
```bash
chmod +x scripts/start_frontend_dev.sh
./scripts/start_frontend_dev.sh
```

**Windows**: 双击 `.bat` 文件或在 CMD 中运行。

### Q: 端口被占用？

修改以下文件中的端口配置：

- Mock API: `scripts/mock_api_server.py` (第 323 行)
- 前端: `magic-school-frontend/vite.config.ts`

### Q: 依赖安装失败？

尝试使用国内镜像：

```bash
npm install --registry=https://registry.npmmirror.com
```

### Q: 前端页面空白？

1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 查看 Network 标签，检查 API 请求是否成功

## 📱 调试提示

### 查看网络请求

1. 打开浏览器开发者工具（F12）
2. 切换到 Network 标签
3. 刷新页面，查看所有 API 请求

### 查看控制台日志

1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签
3. 查看错误信息和警告

### 修改 API 地址

编辑 `magic-school-frontend/.env`:

```bash
VITE_API_BASE_URL=http://localhost:3000/api/v1
VITE_WS_URL=ws://localhost:8765
```
