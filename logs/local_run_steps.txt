========================================
在本地运行前端应用 - 操作步骤
========================================

当前状态: ✓ 已在项目目录
检查: ✓ node_modules 目录存在（依赖已安装）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 立即开始（3 步）:

  步骤 1: 配置环境变量
  
    创建 .env.development 文件:
    
    cat > .env.development << 'EOF'
    # 后端智能体 API 地址
    VITE_BACKEND_URL=http://101.126.128.57:5000
    
    # Mock API 地址
    VITE_API_BASE_URL=http://101.126.128.57:3000/api/v1
    
    # WebSocket 地址
    VITE_WS_URL=ws://101.126.128.57:8765
    EOF

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  步骤 2: 启动开发服务器
    
    npm run dev

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  步骤 3: 访问应用
    
    在浏览器打开:
    http://localhost:5173/

📝 其他有用命令:

  # 如果依赖未安装
  npm install
  
  # 指定端口启动
  npm run dev -- --port 5174
  
  # 构建生产版本
  npm run build
  
  # 预览构建结果
  npm run preview

⚠️ 注意:

  1. 确保 Node.js 已安装 (node -v)
  2. 确保后端服务在远程服务器上运行
  3. 如果无法连接后端，检查网络和防火墙
