========================================
Mac 访问服务器前端界面 - 快速指南
========================================

方法 1: SSH 端口转发（推荐）

  1. 打开 Mac 终端（Command + 空格，输入"终端"）
  
  2. 运行以下命令（将 username 替换为你的服务器用户名）:
  
     ssh -L 5173:localhost:5173 username@101.126.128.57
  
  3. 输入密码
  
  4. 保持终端窗口打开，在浏览器访问:
  
     http://localhost:5173/
  
  5. 使用测试账号登录:
     学生: student / password123
     家长: parent / password123

方法 2: 后台运行 SSH 隧道

  ssh -N -f -L 5173:localhost:5173 username@101.126.128.57
  
  访问: http://localhost:5173/
  
  查看隧道: ps aux | grep ssh
  关闭隧道: killall ssh

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法 3: 使用不同端口

  如果 5173 端口被占用:
  
  ssh -L 8080:localhost:5173 username@101.126.128.57
  
  访问: http://localhost:8080/

常用命令:

  # 建立隧道
  ssh -L 5173:localhost:5173 username@101.126.128.57
  
  # 测试连接
  curl http://localhost:5173
  
  # 关闭隧道（在终端按 Ctrl+C）
  
  # 查看后台隧道
  ps aux | grep ssh

常见问题:

  Q: 连接被拒绝？
  A: 检查 SSH 服务是否运行，确认服务器 IP 正确
  
  Q: 权限被拒绝？
  A: 使用密码登录: ssh -o PreferredAuthentications=password username@101.126.128.57
  
  Q: 端口被占用？
  A: 使用其他端口，如 8080, 8081 等
  
  Q: 连接超时？
  A: 检查网络连接和防火墙设置
