# 🚀 扣子平台快速部署指南

## ⚡ 5 分钟快速部署（推荐）

### 前提条件

- ✅ 阿里云 ECS 服务已运行
- ✅ 服务可通过公网访问
- ✅ 扣子平台账号

---

## 步骤 1：测试后端服务（1 分钟）

```bash
# 替换为您的服务器 IP
curl http://101.126.128.57:5000/health

# 应该返回：
# {"status":"ok","message":"Service is running"}
```

如果失败，检查：
- 阿里云安全组是否开放 5000 端口
- 服务是否正常运行

---

## 步骤 2：在扣子平台创建 Bot（2 分钟）

### 2.1 创建 Bot

1. 访问：https://www.coze.cn/
2. 点击"创建 Bot"
3. 填写信息：
   - **Bot 名称**: 魔法课桌学习助手
   - **Bot 描述**: 小学生学习管理智能体
4. 点击"创建"

### 2.2 配置基本信息

**人设**: 魔法课桌导师
**简介**: 用魔法般有趣的方式帮助小学生管理学习、记录运动、收集成就

**开场白**:
```
✨ 欢迎来到魔法课桌！

我是你的专属魔法课桌导师，可以用魔法般有趣的方式帮助你管理学习、记录运动、收集成就！

需要什么帮助呢？🪄
```

---

## 步骤 3：添加自定义插件（2 分钟）

### 3.1 创建插件

1. 点击"插件" → "添加插件" → "自定义插件"
2. 填写信息：
   - **插件名称**: 魔法课桌 Agent
   - **插件描述**: 调用魔法课桌智能体后端服务

### 3.2 创建工具

**工具配置**：

- **工具名称**: 对话
- **工具描述**: 与魔法课桌智能体对话
- **请求方式**: POST
- **请求地址**: `http://101.126.128.57:5000/run`
- **请求头**:
  ```json
  {
    "Content-Type": "application/json"
  }
  ```
- **请求体**:
  ```json
  {
    "message": "{{message}}",
    "session_id": "{{session_id}}",
    "user_id": "{{user_id}}"
  }
  ```

**输入参数**：
- `message`: 用户消息（必填）
- `session_id`: 会话ID（必填）
- `user_id`: 用户ID（必填）

---

## 步骤 4：测试 Bot（2 分钟）

1. 点击"调试"
2. 输入测试消息：
   ```
   你好
   ```
3. 检查响应是否正常

---

## 步骤 5：发布 Bot（1 分钟）

1. 点击"发布"
2. 填写发布信息
3. 提交审核

## 🔧 常见问题

### Q: 插件调用失败？

A: 检查：
- 服务地址是否正确
- 端口是否开放
- 网络是否通畅

### Q: Bot 回复异常？

A: 检查：
- 输入参数是否正确
- 后端日志是否有错误
