## 🔍 发现的问题

### 🔴 严重问题（P0）

#### 1. 工具参数不一致 - 导致数据隔离失效

**问题描述**:
- 大部分工具使用 `student_name` 作为参数，而不是 `student_id` 或 `user_id`
- 这导致无法与 `auth.users` 表关联，无法实现真正的多用户数据隔离

**影响范围**:
- `tools/homework_db_tool.py` - 所有作业相关工具
- `tools/course_db_tool.py` - 所有课程相关工具
- `tools/exercise_db_tool.py` - 所有运动记录工具
- `tools/achievement_db_tool.py` - 所有成就相关工具

**示例代码**:
```python
# ❌ 错误的实现
def add_homework(student_name: str, title: str, ...):
    student = student_mgr.get_student_by_name(db, student_name)
    # 如果两个学生同名，会导致数据混乱

# ✅ 正确的实现
def add_homework(student_id: int, title: str, runtime: ToolRuntime):
    ctx = runtime.context
    user_id = ctx.get("configurable").get("user_id")
    # 验证 user_id 是否有权操作该 student_id
    if not check_user_access(user_id, user_role, student_id):
        return "错误：无权访问该学生的数据"
```

#### 2. 业务表未初始化

**问题描述**:
- 业务表（students, courses, homeworks 等）没有初始化脚本
- 如果数据库为空，这些表不存在

**影响范围**:
- 所有业务功能无法使用

#### 3. students 表与 auth.users 未关联

**问题描述**:
- `students` 表没有 `user_id` 外键
- 无法通过用户ID查询学生信息

**影响范围**:

```sql
-- 创建索引
CREATE INDEX idx_students_user_id ON students(user_id);

-- 创建触发器：自动更新 updated_at
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = NOW();
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER update_students_updated_at
    BEFORE UPDATE ON students
    FOR EACH ROW
    EXECUTE FUNCTION update_updated_at_column();
```

#### 任务 2.1: 添加权限检查装饰器 - ⏱️ 2 小时

**文件**: `src/auth/decorators.py`
```python
from functools import wraps
from auth.permissions import check_user_access

def require_student_access(func):
    """检查学生访问权限的装饰器"""
    @wraps(func)
    def wrapper(*args, **kwargs):
        runtime = kwargs.get('runtime')
        if not runtime:
            return "错误：缺少运行时上下文"
        
        ctx = runtime.context
        configurable = ctx.get("configurable")
        user_id = configurable.get("user_id")
        user_role = configurable.get("user_role")
        
        # 从参数中获取 student_id
        student_id = kwargs.get('student_id')
        
        if not student_id:
            return "错误：缺少 student_id 参数"
        
        if not check_user_access(user_id, user_role, str(student_id)):
            return "错误：无权访问该学生的数据"
        
        return func(*args, **kwargs)
    
    return wrapper
```
