# 在本地运行魔法课桌前端应用

## 📋 前提条件

确保你的 Mac 已安装以下软件：

- Node.js (推荐 v16 或更高)
- npm (Node.js 自带)
- 任意浏览器（Chrome, Safari, Firefox）

### 检查 Node.js 版本

在终端运行：
```bash
node -v
npm -v
```

如果未安装，访问 https://nodejs.org/ 下载安装。

---

## 🚀 快速开始

### 步骤 1: 安装依赖

在项目根目录运行：
```bash
cd magic-school-frontend
npm install
```

这将安装所有必需的依赖包。

### 步骤 2: 配置环境变量

创建 `.env.development` 文件：

```bash
# 创建环境变量文件
touch .env.development
```

编辑 `.env.development` 文件，添加以下内容：
```env
# 后端智能体 API 地址（远程服务器）
VITE_BACKEND_URL=http://101.126.128.57:5000

# Mock API 地址（远程服务器）
VITE_API_BASE_URL=http://101.126.128.57:3000/api/v1

# WebSocket 地址（如果需要）
VITE_WS_URL=ws://101.126.128.57:8765
```

### 步骤 3: 启动开发服务器

```bash
npm run dev
```

你会看到类似这样的输出：
```
VITE v5.4.21  ready in 412 ms

➜  Local:   http://localhost:5173/
➜  Network: http://192.168.x.x:5173/
```

### 步骤 4: 访问应用

在浏览器中打开：
```
http://localhost:5173/
```

### 步骤 5: 登录测试

使用测试账号登录：
- 学生: `student` / `password123`
- 家长: `parent` / `password123`

## 🛠️ 常见问题

### Q1: npm install 报错

**错误**: `EACCES` 或权限问题

**解决**:
```bash
# 使用 sudo（不推荐）
sudo npm install

# 或使用 nvm（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
npm install
```

### Q2: 端口 5173 被占用

**错误**: `Port 5173 is in use`

**解决**:
```bash
# 方法 1: 停止占用端口的进程
lsof -ti:5173 | xargs kill -9

# 方法 2: 使用其他端口
npm run dev -- --port 5174
```

### Q3: 无法连接到后端

**原因**: 跨域或网络问题

**解决**:
1. 检查后端服务是否运行
2. 检查防火墙设置
3. 使用 Vite 代理配置（已内置在 `vite.config.ts`）

### Q4: 样式显示异常

**解决**:
```bash
# 清除缓存
rm -rf node_modules package-lock.json .vite
npm install
npm run dev
```

### Q5: 浏览器缓存问题

**解决**:
- Chrome: 按 `Cmd + Shift + R` 硬刷新
- Safari: 按 `Cmd + Option + R`
- Firefox: 按 `Cmd + Shift + R`

## 🎨 自定义配置

### 修改端口

编辑 `vite.config.ts`:
```typescript
server: {
  port: 5173,  // 修改为你想要的端口
  host: '0.0.0.0',
}
```

### 添加代理

编辑 `vite.config.ts`:
```typescript
proxy: {
  '/api': {
    target: 'http://101.126.128.57:3000',
    changeOrigin: true
  },
  '/backend': {
    target: 'http://101.126.128.57:5000',
    changeOrigin: true
  }
}
```

### 修改环境变量

创建 `.env.local` 文件（优先级最高）：
```env
VITE_API_BASE_URL=http://your-backend-server.com/api
VITE_BACKEND_URL=http://your-backend-server.com/backend
```

## 🔍 调试技巧

### 1. 查看网络请求

1. 打开浏览器开发者工具（F12 或 Cmd + Option + I）
2. 切换到「Network」标签
3. 筛选 API 请求
4. 查看请求和响应详情

### 2. 查看控制台日志

1. 打开浏览器开发者工具
2. 切换到「Console」标签
3. 查看错误和警告信息

安装浏览器扩展：
- Chrome: React Developer Tools
- Firefox: React Developer Tools

### 4. 查看源码映射

在 `vite.config.ts` 中启用：
```typescript

  sourcemap: true
}
```


### 预览构建结果

```bash
npm run preview
```

### 部署到服务器

将 `dist/` 目录上传到服务器，然后使用 Nginx 或 Apache 配置：

**Nginx 配置示例**:
```nginx
server {
    listen 80;
    server_name your-domain.com;

    root /path/to/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://backend-server:5000;
        proxy_set_header Host $host;
    }
}
```
