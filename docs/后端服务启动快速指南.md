# 后端服务启动 - 快速指南

## 🚀 一键启动（推荐）

### Linux/macOS

```bash
cd /workspace/projects
bash scripts/start_all_services.sh
```

### Windows

```cmd
cd scripts
start_all_services.bat
```

---

## ⚡ 最小化启动（仅智能体）

```bash
cd /workspace/projects
bash scripts/http_run.sh -p 5000
```

---

## 📋 服务列表

| 服务 | 端口 | 必需 | 命令 |
|------|------|------|------|
| 智能体 API | 5000 | ✅ 必需 | `bash scripts/http_run.sh -p 5000` |
| Mock API | 3000 | ⚠️ 可选 | `python3 scripts/mock_api_server.py` |
| WebSocket | 8765 | ⚠️ 可选 | `python3 src/websocket_server.py` |

---

## ✅ 验证服务

```bash
# 检查进程
ps aux | grep python

# 检查端口
netstat -tlnp | grep -E "(5000|3000|8765)"

# 测试 API
curl http://localhost:5000/docs
```

---

## 🛑 停止服务

```bash
# 停止所有服务
pkill -f "python.*main.py"
pkill -f "python.*mock_api"
pkill -f "python.*websocket"
```

---

## 📊 访问地址

- 智能体 API: http://localhost:5000/docs
- Mock API: http://localhost:3000/docs
- 前端界面: http://localhost:5173

---

## ❓ 常见问题

**端口被占用？**
```bash
lsof -ti:5000 | xargs kill -9
```

**查看日志？**
```bash
tail -f /app/work/logs/bypass/app.log
```
