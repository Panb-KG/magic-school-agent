#### 1.2 学生访问权限检查
```python
def check_student_access(runtime: ToolRuntime, student_id: int) -> bool:
    """检查用户是否有权访问该学生的数据"""
```

**检查逻辑**:
1. **学生角色**: 只能访问自己的数据
   - 通过 `student.user_id == current_user_id` 验证
   
2. **家长角色**: 只能访问关联学生的数据
   - 通过 `permissions_manager.can_access_student()` 验证

3. **安全策略**:
   - 检查失败默认拒绝访问
   - 详细的权限拒绝日志
   - 无效参数自动拒绝

#### 1.3 权限装饰器系统

##### 装饰器 1: `@require_student_access()`
**用途**: 要求学生数据访问权限

```python
@tool
@require_student_access()
def get_homework_list(student_id: int, ..., runtime: ToolRuntime):
    # 自动检查权限，无权则返回错误信息
    ...
```

**特性**:
- ✅ 自动检查 `student_id` 参数
- ✅ 自动返回错误信息
- ✅ 支持自定义参数名
- ✅ 支持自定义错误信息

##### 装饰器 2: `@require_any_permission()`
**用途**: 要求拥有任意一个权限

```python
@tool
@require_any_permission('view_student_data', 'view_own_data')
def get_student_info(student_id: int, ...):
    ...
```

##### 装饰器 3: `@require_all_permissions()`
**用途**: 要求拥有所有权限

```python
@tool
@require_all_permissions('view_student_data', 'edit_homework')
def modify_homework(student_id: int, homework_id: int, ...):
    ...
```

##### 装饰器 4: `@require_role()`
**用途**: 要求特定角色

```python
@tool
@require_role('parent')
def parent_view_dashboard(...):
    ...
```

##### 装饰器 5: `@log_access()`
**用途**: 记录访问日志

```python
@tool
@log_access
def get_homework_list(student_id: int, ...):
    ...
```

**日志内容**:
- 函数名
- 用户ID和角色
- 会话ID和线程ID
- 调用参数

## ⚠️ 注意事项

### 1. 装饰器顺序

**正确顺序**:
```python
@tool
@require_student_access()  # 装饰器在 @tool 之后
def function(...):
    ...
```

**错误顺序**:
```python
@require_student_access()  # ❌ 装饰器在 @tool 之前
@tool
def function(...):
    ...
```

### 2. 参数命名

**必须使用 `student_id`**:
```python
def get_data(student_id: int, ...):  # ✅ 正确
    ...

def get_data(stu_id: int, ...):      # ❌ 错误
    ...
```

**或指定参数名**:
```python
@require_student_access(student_id_param='stu_id')
def get_data(stu_id: int, ...):
    ...
```

### 3. runtime 参数

**必须包含 `runtime` 参数**:
```python
def get_data(student_id: int, runtime: ToolRuntime):  # ✅ 正确
    ...

def get_data(student_id: int):                      # ❌ 错误
    ...
```

### 4. 错误处理

**装饰器自动处理错误**:
- ✅ 权限拒绝 → 自动返回错误信息
- ✅ 参数缺失 → 自动返回错误信息
- ✅ 异常捕获 → 安全执行
