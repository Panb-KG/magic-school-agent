## 📝 关键技术点

### 1. 权限检查机制

```python
def check_student_access(runtime, student_id):
    user_id, user_role = get_user_context(runtime)
    
    if user_role == 'student':
        # 学生只能访问自己的数据
        student = student_mgr.get_student_by_id(db, student_id)
        return student.user_id == user_id
    
    elif user_role == 'parent':
        # 家长可以访问关联学生的数据
        return permissions_manager.can_access_student(user_id, student_id)
    
    return False
```

### 2. 数据库关联

```sql
-- students 表通过 user_id 关联到 auth.users

FOREIGN KEY (user_id)
```

## ⚠️ 注意事项

### 1. 数据迁移

如果已有数据，需要执行数据迁移：

```sql
-- 为现有学生创建 user_id
UPDATE students s
SET user_id = (
    SELECT user_id 
    FROM auth.users u 
    WHERE u.student_name = s.name
    LIMIT 1
)
WHERE user_id IS NULL;
```
