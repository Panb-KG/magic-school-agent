根据部署方式选择测试入口：

#### 方式一：扣子平台测试
- **访问地址**: https://coze.cn
- **要求**: 登录扣子账号
- **Bot ID**: 需要在个人中心获取

#### 方式二：自建服务器测试
- **访问地址**: https://your-domain.com
- **要求**: 需要登录账号
- **认证方式**: JWT Token

#### 方式三：API 测试
- **API 地址**: https://your-domain.com/api/chat
- **要求**: 获取 API Token
- **工具**: Postman / curl

### 2. 测试环境检查

测试前请确认：
- [ ] 智能体已成功部署
- [ ] 数据库连接正常
- [ ] 大模型 API 配置正确
- [ ] 文件存储服务可用
- [ ] WebSocket 连接正常

### 3. 测试工具准备

推荐工具：
- **浏览器**: Chrome / Firefox / Safari

## 🌐 WebSocket 实时对话测试

### WebSocket 连接

```javascript
// JavaScript 示例
const ws = new WebSocket('wss://your-domain.com/ws/chat');

ws.onopen = () => {
  console.log('WebSocket 连接已建立');

  // 发送消息
  ws.send(JSON.stringify({
    type: 'message',
    message: '你好，请自我介绍一下',
    user_id: 'user123',
    student_id: 1
  }));
};

ws.onmessage = (event) => {
  const response = JSON.parse(event.data);
  console.log('收到回复:', response);

  if (response.type === 'agent_reply') {
    console.log('Agent 回复:', response.content);
  }
};

ws.onerror = (error) => {
  console.error('WebSocket 错误:', error);
};

ws.onclose = () => {
  console.log('WebSocket 连接已关闭');
};
```
