## 🔀 数据流程图

### 1. 用户登录流程

```
用户输入账号密码
      │
      ▼
前端调用 POST /api/v1/auth/login
      │
      ▼
┌─────────────────┐
│  Mock API       │
│  Server         │
└────────┬────────┘
         │ 验证用户
         │ 生成 JWT Token
         ▼
    返回 Token + 用户信息
         │
         ▼
前端存储到 localStorage
         │
         ▼
后续请求自动携带 Authorization: Bearer {token}
```

### 2. 智能体对话流程

```
用户发送消息
      │
      ▼
前端调用 POST /stream_run
      │
      ├─ Authorization: Bearer {token}
      ├─ Body: {
      │     query: "用户的问题",
      │     session_id: "会话ID",
      │     user_id: "用户ID",
      │     user_role: "student/parent"
      │   }
      ▼
┌─────────────────┐
│  智能体 API     │
│  Server         │
│  Port: 5000     │
└────────┬────────┘
         │ 验证 Token
         │ 提取用户信息
         ▼
┌─────────────────┐
│  LangGraph      │
│  Agent          │
└────────┬────────┘
         │ 分析问题
         │ 调用工具:
         │ - get_current_time (获取时间)
         │ - get_student_info (查询学生信息)
         │ - get_homework_list (查询作业)
         │ - 等等...
         ▼
    生成回复
         │
         ▼
SSE 流式返回响应
 event: message
 data: {"content": "部分"}
      │
      ▼
前端实时更新 UI
```

### 3. 实时数据更新流程

```
后端数据变化
(如: 作业状态更新)
      │
      ▼
┌─────────────────┐
│  WebSocket      │
│  Server         │
└────────┬────────┘
         │ 推送消息
         │ {
         │   type: "homework_update",
         │   channel: "homework",
         │   data: {...}
         │ }
         ▼
前端 WebSocket
      │ on('message')
      ▼
更新 UI 界面
```

## 📡 WebSocket 连接流程

```
┌──────────┐
│  前端    │
└────┬─────┘
     │
     │ 1. 建立 WebSocket 连接
     │    io(WS_URL)
     ▼
┌──────────────────┐
│   WebSocket      │
│   Server         │
│   Port: 8765     │
└────┬─────────────┘
     │
     │ 2. 连接成功
     │    emit('connect')
     ▼
┌──────────────────┐
│  前端            │
│  on('connect')   │
└────┬─────────────┘
     │
     │ 3. 订阅频道
     │    emit('subscribe', { channels: ['dashboard', 'chat'] })
     ▼
┌──────────────────┐
│   WebSocket      │
│   Server         │
└────┬─────────────┘
     │
     │ 4. 保存订阅关系
     │    subscriptions[client] = ['dashboard', 'chat']
     ▼
┌──────────────────┐
│   后端数据变化   │
└────┬─────────────┘
     │
     │ 5. 推送消息
     │    broadcast_to_channel('dashboard', { data: {...} })
     ▼
┌──────────────────┐
│   WebSocket      │
│   Server         │
└────┬─────────────┘
     │
     │ 6. 发送给订阅该频道的客户端
     ▼
┌──────────────────┐
│  前端            │
│  on('message')   │
└────┬─────────────┘
     │
     │ 7. 更新 UI
     ▼
┌──────────────────┐
│   页面界面       │
└──────────────────┘
```

## 🔄 完整请求生命周期

### HTTP 请求生命周期

```
┌─────────────────────────────────────────────────────────────┐
│  1. 前端发起请求                                            │
│     axios.get('/api/v1/dashboard')                          │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  2. Axios 请求拦截器                                         │
│     - 添加 Authorization: Bearer {token}                    │
│     - 设置 Content-Type: application/json                   │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  3. Vite 代理处理                                            │
│     - /api/* → http://localhost:3000                        │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  4. Mock API Server 接收请求                                 │
│     FastAPI                                                  │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  5. 验证 Token                                               │
│     - 解析 JWT                                               │
│     - 验证过期时间                                            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  6. 业务逻辑处理                                              │
│     - 查询数据库/返回模拟数据                                 │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  7. 返回响应                                                  │
│     HTTP 200 OK                                              │
│     { success: true, data: {...} }                          │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  8. Axios 响应拦截器                                          │
│     - 提取 response.data                                     │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  9. 前端更新 UI                                              │
│     - 显示数据                                                │
└─────────────────────────────────────────────────────────────┘
```

### SSE 流式响应生命周期

```
┌─────────────────────────────────────────────────────────────┐
│  1. 前端发起流式请求                                          │
│     fetch('/stream_run', { ... })                            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  2. 智能体 API Server 接收请求                                │
│     FastAPI + SSE                                            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  3. 调用 LangGraph Agent                                      │
│     agent.stream(input, ...)                                 │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  4. Agent 执行                                                │
│     - 分析问题                                                │
│     - 调用工具                                                │
│     - 生成回复                                                │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  5. 流式返回数据块                                            │
│     event: message\n                                         │
│     data: {"content": "部分内容"}\n\n                        │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  6. 前端读取数据流                                            │
│     response.body.getReader()                                │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  7. 实时更新 UI                                              │
│     - 逐步显示文本                                            │
│     - 提供流畅体验                                            │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  8. 流结束                                                    │
│     data: {"type": "end"}\n\n                                │
└─────────────────────────────────────────────────────────────┘
```

## 📊 端口分配

| 服务 | 端口 | 协议 | 用途 |
|------|------|------|------|
| 前端开发服务器 | 5173 | HTTP | React 应用开发 |
| Mock API 服务器 | 3000 | HTTP | 开发调试数据接口 |
| 智能体 API 服务器 | 5000 | HTTP/SSE | Agent 对话接口 |
| 多用户 API 服务器 | 待定 | HTTP | 用户认证和管理 |
| WebSocket 服务器 | 8765 | WebSocket | 实时消息推送 |
| PostgreSQL 数据库 | 5432 | TCP | 结构化数据存储 |
| 对象存储服务 | 待定 | HTTP/S3 | 文件存储 |

## 🔗 环境变量配置

### 开发环境 (.env.development)

```env
# 前端
VITE_API_BASE_URL=http://localhost:3000/api/v1
VITE_BACKEND_URL=http://localhost:5000
VITE_WS_URL=ws://localhost:8765
```

### 生产环境 (.env.production)

```env
# 前端
VITE_API_BASE_URL=https://your-domain.com/api
VITE_BACKEND_URL=https://your-domain.com/backend
VITE_WS_URL=wss://your-domain.com/ws
```
