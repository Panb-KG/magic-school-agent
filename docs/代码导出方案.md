# 代码导出方案 - 扣子IDE 到 阿里云ECS

| 方案 | 适用系统 | 难度 | 说明 |
|------|---------|------|------|

## 方案1：扣子IDE直接下载（推荐，最简单）

### 步骤1：在扣子IDE中打包代码

```bash
# 在扣子IDE终端执行
cd /workspace
tar -czf magic-school-project.tar.gz projects/

# 验证打包成功
ls -lh magic-school-project.tar.gz
```

### 步骤2：下载到本地

**方法A - 通过扣子IDE界面**:
1. 在文件管理器中找到 `magic-school-project.tar.gz`
2. 右键点击 → 下载
3. 保存到您的电脑

**方法B - 如果扣子IDE支持Web终端下载**:
```bash
# 部分IDE支持通过Web下载
# 查看IDE文档了解如何下载文件
```

### 步骤3：上传到ECS

**方法A - 使用阿里云ECS控制台上传**:
1. 登录阿里云ECS控制台
2. 找到您的实例
3. 点击「远程连接」→「VNC连接」
4. 在VNC界面中：
   ```bash
   # 创建临时目录
   mkdir -p /tmp/upload
   cd /tmp/upload

   # 在VNC界面中，找到文件上传按钮
   # 将 magic-school-project.tar.gz 上传到此目录
   ```

**方法B - 如果您有其他SSH工具**:
使用该工具上传文件（见方案2、3、5）

### 步骤4：在ECS上解压

```bash
# SSH登录到ECS
ssh root@your-ecs-ip

# 移动上传的文件到项目目录
mv /tmp/upload/magic-school-project.tar.gz /workspace/

# 解压
cd /workspace
tar -xzf magic-school-project.tar.gz

# 验证解压成功
ls -la projects/
```

## 方案2：使用 WinSCP（Windows用户推荐）

### 步骤1：在扣子IDE中打包代码

```bash
cd /workspace
tar -czf magic-school-project.tar.gz projects/
```

### 步骤2：下载 WinSCP

访问：https://winscp.net/eng/download.php
下载并安装 WinSCP。

### 步骤3：配置 WinSCP

1. 打开 WinSCP
2. 新建站点：
   - **文件协议**: SFTP
   - **主机名**: 您的ECS公网IP
   - **端口**: 22
   - **用户名**: root
   - **密码**: 您的ECS密码

3. 点击「登录」

### 步骤4：上传文件

1. 在左侧本地窗口中，找到 `magic-school-project.tar.gz`
2. 在右侧远程窗口中，导航到 `/workspace`
3. 拖拽文件到右侧窗口

### 步骤5：在ECS上解压

```bash
cd /workspace
tar -xzf magic-school-project.tar.gz
ls -la projects/
```

## 方案3：安装 OpenSSH（Mac/Linux用户）

### Mac 用户

```bash
# 检查是否已安装
which scp

# 如果没有安装，安装 Homebrew（如果没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 openssh
brew install openssh

# 验证安装
which scp
```

### Linux 用户（Ubuntu/Debian）

```bash
# 安装 openssh-client
sudo apt update
sudo apt install openssh-client -y

# 验证安装
which scp
```

### Linux 用户（CentOS/RHEL）

```bash
# 安装 openssh-clients
sudo yum install openssh-clients -y

# 验证安装
which scp
```

### 使用 SCP 上传

```bash
# 在您的电脑上执行
scp /path/to/magic-school-project.tar.gz root@your-ecs-ip:/workspace/

# 然后SSH登录到ECS
ssh root@your-ecs-ip

# 解压
cd /workspace
tar -xzf magic-school-project.tar.gz
```

## 方案4：使用 Git（如果代码已上传到Git仓库）

### 步骤1：在扣子IDE中推送到Git

```bash
# 如果已经有Git仓库
cd /workspace/projects
git add .
git commit -m "Initial commit"
git push origin main
```

### 步骤2：在ECS上克隆

```bash
# SSH登录到ECS
ssh root@your-ecs-ip

# 安装Git（如果没有）
sudo apt install git -y  # Ubuntu/Debian
# 或
sudo yum install git -y  # CentOS/RHEL

# 克隆仓库
cd /workspace
git clone https://github.com/your-username/magic-school.git

# 重命名
mv magic-school projects
```

### 步骤3：安装依赖

```bash
cd /workspace/projects
pip3 install -r requirements.txt
cd magic-school-frontend
npm install
```

## 方案5：使用其他SFTP工具

### 选项A：FileZilla（跨平台）

1. 下载 FileZilla: https://filezilla-project.org/
2. 配置站点：
   - 主机: 您的ECS IP
   - 用户: root
   - 密码: 您的ECS密码
   - 端口: 22
3. 连接后上传文件

### 选项B：Cyberduck（Mac）

1. 下载 Cyberduck: https://cyberduck.io/
2. 配置SFTP连接
3. 上传文件

### 选项C：Termius（跨平台）

1. 下载 Termius: https://termius.com/
2. 配置SFTP
3. 使用文件传输功能

## 方案6：使用阿里云VNC控制台（无需额外工具）

### 步骤1：打包代码

在扣子IDE终端执行：
```bash
cd /workspace
tar -czf magic-school-project.tar.gz projects/
```

### 步骤2：下载到本地

根据扣子IDE的文件下载功能下载 `magic-school-project.tar.gz`

### 步骤3：使用阿里云VNC上传

1. 登录阿里云ECS控制台
2. 找到您的实例
3. 点击「远程连接」→「VNC连接」

4. 在VNC窗口中：
   ```bash
   # 创建上传目录
   mkdir -p /tmp/upload
   cd /tmp/upload

   # 点击VNC窗口顶部的「文件」或「上传」按钮
   # 上传 magic-school-project.tar.gz
   ```

5. 上传完成后：
   ```bash
   # 移动文件
   mv /tmp/upload/magic-school-project.tar.gz /workspace/

   # 解压
   cd /workspace
   tar -xzf magic-school-project.tar.gz

   # 验证
   ls -la projects/
   ```

## 🆘 常见问题

### Q1: 扣子IDE如何下载文件？
**A**: 在扣子IDE的文件管理器中，右键点击文件，选择"下载"。

### Q2: VNC上传文件失败怎么办？
**A**: 检查文件大小是否超过限制（通常限制100MB）。如果超过，先在本地压缩或分卷压缩。

### Q3: 执行 tar 命令时提示找不到命令？
**A**: 说明打包在扣子IDE中完成，下载的文件已经是压缩包了。在ECS上只需执行 tar 解压命令。

### Q4: 上传后文件损坏？
**A**: 使用 `md5sum` 或 `sha256sum` 校验文件完整性：
```bash
# 本地
md5sum magic-school-project.tar.gz

# ECS
md5sum /workspace/magic-school-project.tar.gz

# 两个值应该相同
```
