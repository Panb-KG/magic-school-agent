## 🔌 API 接口说明

### 核心接口

后端智能体提供两个主要接口：

#### 1. 非流式对话：`POST /run`

**请求**:
```json
{
  "query": "用户的问题",
  "session_id": "会话ID",
  "user_id": "用户ID",
  "user_role": "student"
}
```

**响应**:
```json
{
  "output": "智能体的回复",
  "session_id": "会话ID",
  "run_id": "运行ID",
  "tools_used": ["get_current_time"],
  "time_cost_ms": 1500
}
```

#### 2. 流式对话：`POST /stream_run`

**请求**: 与非流式相同

**响应** (SSE 格式):
```
event: message
data: {"content": "部分回复"}

event: message
data: {"content": "后续内容"}

event: message
data: {"type": "end"}
```

### 示例 3: 使用服务层 API

```tsx
import React, { useState } from 'react';
import { Button, Input } from 'antd';
import { streamMessage, generateSessionId } from '@/services/chatService';

const DirectApiPage: React.FC = () => {
  const [input, setInput] = useState('');
  const [response, setResponse] = useState('');
  const [loading, setLoading] = useState(false);
  const [sessionId] = useState(generateSessionId());

  const handleSend = async () => {
    if (!input.trim()) return;
    
    setLoading(true);
    setResponse('');
    
    try {
      await streamMessage(
        input,
        sessionId,
        // 收到内容块
        (chunk) => {
          setResponse(prev => prev + chunk);
        },
        // 完成
        (fullResponse) => {
          console.log('完整回复:', fullResponse);
        },
        // 错误
        (error) => {
          console.error('错误:', error);
        }
      );
    } catch (error) {
      console.error('发送失败:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div style={{ padding: 24 }}>
      <Input.TextArea
        value={input}
        onChange={(e) => setInput(e.target.value)}
        placeholder="输入你的问题..."
        autoSize={{ minRows: 3, maxRows: 6 }}
      />
      <Button 
        type="primary" 
        onClick={handleSend}
        loading={loading}
        style={{ marginTop: 16 }}
      >
        发送
      </Button>
      <div style={{ marginTop: 16, padding: 16, background: '#f5f5f5' }}>
        <h4>回复：</h4>
        <p>{response || '等待回复...'}</p>
      </div>
    </div>
  );
};

export default DirectApiPage;
```

## ❓ 常见问题

### Q1: 发送消息后没有响应

**可能原因**:
1. 后端服务未启动
2. 环境变量配置错误
3. Token 无效或过期

**解决方法**:
```bash
# 1. 检查后端服务
ps aux | grep python | grep main.py

# 2. 检查端口
netstat -tlnp | grep 5000

# 3. 检查环境变量
cat magic-school-frontend/.env.development

# 4. 清除 Token 并重新登录
localStorage.clear()
```

### Q2: 流式响应卡住

**可能原因**:
1. 网络连接中断
2. 后端执行超时
3. Token 过期

**解决方法**:
1. 检查网络连接
2. 增加超时时间
3. 重新登录获取新 Token

### Q3: 提示 "401 Unauthorized"

**原因**: Token 无效或过期

**解决方法**:
```tsx
// 自动跳转到登录页（已内置）
// 或手动处理
localStorage.clear();
window.location.href = '/login';
```

### Q4: 如何自定义智能体的行为？

修改系统提示词：

编辑 `config/agent_llm_config.json`:

```json
{
  "sp": "你是一个友好的学习助手...",
  "config": {
    "model": "doubao-seed-1-6-251015",
    "temperature": 0.7
  }
}
```

### Q5: 如何添加新工具？

1. 在 `src/tools/` 目录下创建工具文件
2. 在 `src/agents/agent.py` 中导入并注册工具
3. 重启后端服务

## 💡 高级用法

### 添加消息持久化

```tsx
import { useEffect } from 'react';
import { authStorage } from '@/contexts/AuthContext';

const ChatWithPersistence: React.FC = () => {
  const [messages, setMessages] = useState<ChatMessage[]>([]);

  // 加载历史消息
  useEffect(() => {
    const history = localStorage.getItem('chat_history');
    if (history) {
      setMessages(JSON.parse(history));
    }
  }, []);

  // 保存消息
  const handleSendMessage = async (message: string) => {
    const newMessage: ChatMessage = {
      id: Date.now().toString(),
      role: 'user',
      content: message,
      timestamp: new Date().toISOString(),
    };

    setMessages(prev => {
      const updated = [...prev, newMessage];
      localStorage.setItem('chat_history', JSON.stringify(updated));
      return updated;
    });
  };

  return <MagicChat onMessageSent={handleSendMessage} />;
};
```

### 添加消息导出功能

```tsx
const exportMessages = () => {
  const dataStr = JSON.stringify(messages, null, 2);
  const dataBlob = new Blob([dataStr], { type: 'application/json' });
  const url = URL.createObjectURL(dataBlob);
  const link = document.createElement('a');
  link.href = url;
  link.download = `chat-history-${Date.now()}.json`;
  link.click();
};

// 在组件中添加导出按钮
<Button onClick={exportMessages}>导出对话</Button>
```

### 添加语音输入

```bash
npm install speech-recognition-polyfill
```

```tsx
import SpeechRecognition from 'speech-recognition-polyfill';

const VoiceInput: React.FC = () => {
  const [isListening, setIsListening] = useState(false);

  const toggleListening = () => {
    if (isListening) {
      SpeechRecognition.stop();
    } else {
      SpeechRecognition.start();
    }
    setIsListening(!isListening);
  };

  return (
    <Button 
      icon={<AudioOutlined />} 
      onClick={toggleListening}
      type={isListening ? 'primary' : 'default'}
    >
      {isListening ? '录音中...' : '语音输入'}
    </Button>
  );
};
```
