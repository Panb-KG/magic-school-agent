## 1. 智能体概述

### 1.1 产品简介

魔法课桌学习助手智能体是一款专为小学生和家长设计的魔法学校主题学习管理智能体。以哈利波特魔法世界为设计灵感，将学习过程游戏化、趣味化，激发孩子的学习兴趣。

### 1.2 设计理念

- **魔法学校风格**：界面与交互采用魔法世界元素，营造沉浸式学习体验
- **引导式教育**：坚持启发式教学原则，引导思考而非直接给出答案
- **安全防护**：严格的内容过滤机制，保护小学生免受不良信息影响
- **家校共育**：支持家长角色监控与管理，实现家校协同教育
- **时间感知**：智能体具备时间魔法能力，所有信息都明确标注时间参照

### 1.3 目标用户

- **主要用户**：小学生（6-12岁）
- **次要用户**：学生家长/监护人

## 2. 功能模块

### 2.1 智能对话中心 🧙‍♂️

#### 核心功能
- **魔法助手聊天**：与小魔法助手进行自然语言对话
- **学习问题咨询**：询问学习问题，获得启发式引导
- **日常对话交互**：进行友好的日常交流
- **长期记忆对话**：基于历史对话的个性化交流

#### 特色
- 小学生友好的对话风格
- 智能时间感知，明确时间参照
- 个性化对话记忆，记住孩子喜好
- 多轮对话能力，理解上下文

### 2.2 历史对话管理 💬

#### 核心功能
- **对话会话列表**：按时间倒序查看所有历史对话会话
- **会话搜索**：通过关键词快速定位特定对话
- **标题自动生成**：智能为对话会话生成摘要标题
- **会话详情查看**：查看完整对话历史记录
- **会话管理**：更新会话标题、删除对话会话等操作
- **批量标题生成**：一键为所有会话生成标题

#### 特色
- 智能标题生成，快速识别对话主题
- 高效搜索功能，快速回溯重要对话
- 按时间倒序排列，最新对话优先展示
- 支持会话恢复，继续之前的对话
- 灵活的会话管理，方便整理和清理

#### API支持
- `POST /api/v1/conversations` - 创建新对话会话
- `GET /api/v1/conversations` - 获取对话会话列表
- `GET /api/v1/conversations/{id}` - 获取会话详情
- `GET /api/v1/conversations/search` - 搜索对话会话
- `PUT /api/v1/conversations/{id}` - 更新会话标题
- `DELETE /api/v1/conversations/{id}` - 删除对话会话
- `POST /api/v1/conversations/{id}/messages` - 添加对话消息
- `POST /api/v1/conversations/{id}/generate-title` - 生成会话标题
- `POST /api/v1/conversations/batch-generate-titles` - 批量生成标题

## 6. 核心特性

### 6.1 时间感知能力 ⏰

智能体具备"时间魔法"能力，在处理课程、作业、文档时自动获取并标注时间信息：

- **当前时间查询**：实时获取当前日期和时间
- **日期范围计算**：自动计算本周、本月日期范围
- **时间标注**：所有信息都明确时间参照
- **时间提醒**：基于时间的智能提醒

**教育策略**：
1. **启发式提问**：通过提问引导思考
2. **思路梳理**：帮助理清解题思路
3. **资源推荐**：推荐学习方法与资料
4. **鼓励式反馈**：正面激励与建设性建议

**示例对话**：
```
❌ 错误："这道题的答案是 42"
✅ 正确："这道题的关键是要理解什么是'平均数'。
         你可以先想想：平均数是怎么计算出来的？
         告诉我你的思路，我帮你看看对不对～"
```

## 7. 部署说明

### 7.1 环境要求

#### 服务器配置
- **操作系统**：Linux (Ubuntu 20.04+ 推荐)
- **CPU**：4 核及以上
- **内存**：8GB 及以上
- **硬盘**：100GB 及以上
- **网络**：公网 IP 或内网穿透

#### 软件依赖
- Python 3.8+
- Node.js 16+
- PostgreSQL 12+
- Redis 6+ (可选，用于缓存)
- Nginx 1.18+

### 7.2 部署步骤

#### 步骤 1：克隆代码
```bash
git clone <repository_url>
cd magic-desk-agent
```

#### 步骤 2：安装后端依赖
```bash
pip install -r requirements.txt
```

#### 步骤 3：配置环境变量
创建 `.env` 文件：
```bash
# 数据库配置
DATABASE_URL=postgresql://user:password@localhost/magic_desk

# 大模型配置
MODEL_API_KEY=your_api_key
MODEL_BASE_URL=https://api.example.com

# 对象存储配置
S3_ENDPOINT=your_s3_endpoint
S3_ACCESS_KEY=your_access_key
S3_SECRET_KEY=your_secret_key
S3_BUCKET=magic-desk

# JWT 密钥
JWT_SECRET_KEY=your_secret_key

# WebSocket 配置
WS_PORT=8001
```

#### 步骤 4：初始化数据库
```bash
python scripts/init_db.py
```

#### 步骤 5：安装前端依赖
```bash
cd frontend
npm install
npm run build
```

#### 步骤 6：配置 Nginx
创建 Nginx 配置文件 `/etc/nginx/sites-available/magic-desk`:
```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /path/to/frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api/ {
        proxy_pass http://localhost:8000/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # WebSocket 代理
    location /ws/ {
        proxy_pass http://localhost:8001/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

#### 步骤 7：启动服务
```bash
# 启动后端 API
python src/main.py

# 启动 WebSocket 服务
python scripts/start_websocket.py
```

#### 步骤 8：配置服务自启（systemd）
创建 `/etc/systemd/system/magic-desk.service`:
```ini
[Unit]
Description=Magic Desk Agent
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/path/to/magic-desk-agent
ExecStart=/usr/bin/python3 src/main.py
Restart=always

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
sudo systemctl enable magic-desk
sudo systemctl start magic-desk
```

## 10. FAQ

### Q1: 如何保护孩子隐私？
**A**: 
- 对话记录查看权限可配置
- 敏感信息加密存储
- 家长可随时删除孩子数据
- 严格遵守数据保护法规

### Q2: 智能体是否会直接给出作业答案？
**A**: 
不会。智能体坚持引导式教育原则，通过启发式提问、思路梳理等方式引导孩子独立思考，培养自主解决问题的能力。

### Q3: 如何切换学生和家长角色？
**A**: 
登录后，点击右上角头像，选择"切换角色"即可在学生和家长模式之间切换。家长模式需要验证家长身份。

### Q4: 支持哪些设备？
**A**: 
支持电脑、平板、手机等设备访问。响应式设计确保在不同设备上都有良好的使用体验。

### Q5: 如何联系技术支持？
**A**: 
- 在线客服：网站右下角"帮助"按钮
- 邮箱：support@magic-desk.com
- 工作时间：周一至周五 9:00-18:00

### Q6: 长期记忆系统会记录什么？
**A**: 
长期记忆系统会记录：
- 用户性格特点
- 兴趣爱好
- 学习偏好
- 知识点掌握情况
- 重要对话摘要

家长可在家长管理中心查看这些信息。

### Q7: 时间感知功能如何工作？
**A**: 
智能体内置时间魔法工具，能够：
- 获取当前日期和时间
- 计算日期范围（本周、本月等）
- 在对话中自动标注时间参照
- 基于时间进行提醒

例如，查询"明天的课程"时，智能体会自动获取当前时间，计算明天的日期，然后查询课程信息。

### Q8: 如何取消账号？
**A**: 
联系客服提交账号注销申请，客服会在3个工作日内处理。注销后所有数据将被永久删除。
