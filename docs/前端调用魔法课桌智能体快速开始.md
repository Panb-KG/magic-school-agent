# 前端调用魔法课桌智能体 - 快速开始

## 🎯 一句话总结

**前端通过调用 `http://localhost:5000/stream_run` 接口，使用流式响应与魔法课桌智能体进行实时对话。**

---

## 🚀 3 步快速上手

### 第 1 步：启动后端服务

```bash
cd /workspace/projects
bash scripts/http_run.sh -p 5000
```

### 第 2 步：配置环境变量

编辑 `magic-school-frontend/.env.development`：

```env
VITE_BACKEND_URL=http://localhost:5000
```

### 第 3 步：使用组件

```tsx
import MagicChat from '@/components/MagicChat';

function App() {
  return <MagicChat />;
}
```

---

## 📦 核心文件

### 1. 服务层 (`src/services/chatService.ts`)

提供两个核心函数：

```typescript
// 非流式对话
const response = await sendMessage('你好');

// 流式对话（推荐）
await streamMessage(
  '你好',
  sessionId,
  (chunk) => console.log('收到:', chunk),  // 实时接收
  (full) => console.log('完成:', full)    // 完成回调
);
```

### 2. 组件层 (`src/components/MagicChat/`)

完整的聊天界面组件：

```tsx
<MagicChat
  height="600px"
  showHeader={true}
  onMessageReceived={(msg) => console.log(msg)}
/>
```

### 3. 页面层 (`src/pages/MagicChatPage/`)

示例页面，展示如何使用组件：

```tsx
import MagicChatPage from '@/pages/MagicChatPage';

// 在路由中配置
{ path: '/chat', element: <MagicChatPage /> }
```

## 🔌 API 接口

### 请求格式

```typescript
POST http://localhost:5000/stream_run
Headers: {
  "Content-Type": "application/json",
  "Authorization": "Bearer {token}"
}
Body: {
  "query": "用户消息",
  "session_id": "会话ID",
  "user_id": "用户ID",
  "user_role": "student/parent"
}
```

### 响应格式（SSE 流式）

```
event: message
data: {"content": "部分回复"}

event: message
data: {"content": "后续内容"}

event: message
data: {"type": "end"}
```

## 🐛 常见问题

**Q: 发送消息没有响应？**

A: 检查后端服务是否启动：
```bash
ps aux | grep python | grep main.py
```

**Q: 提示 401 错误？**

A: 清除缓存并重新登录：
```bash
localStorage.clear()
window.location.href = '/login'
```

**Q: 流式响应卡住？**

A: 检查网络连接和后端日志：
```bash
tail -f /app/work/logs/bypass/app.log
```
