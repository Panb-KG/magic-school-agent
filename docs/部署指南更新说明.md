- 详细的配置说明和注释
- 安全建议和配置提示

**主要配置项**：
```bash
# 数据库配置
DATABASE_URL=

# 认证配置
JWT_SECRET=
JWT_EXPIRE_SECONDS=

# 阿里云百炼配置（必须）

COZE_INTEGRATION_MODEL_BASE_URL=

# 应用配置

LOG_LEVEL=INFO

# 服务器配置
PORT=8000  # 必须配置为8000

# CORS配置
CORS_ORIGINS=*

# 记忆系统配置
MEMORY_STORAGE_TYPE=postgresql

# 速率限制配置
RATE_LIMIT_PER_MINUTE=60
```
