## ⚠️ 注意事项

1. **端口占用**: 如果 5173 端口被占用，Vite 会自动切换到 5174 端口
2. **Mock API**: 当前使用 Mock API 服务器（端口 3000），数据为模拟数据
3. **实时通信**: WebSocket 连接配置在 8765 端口
4. **开发模式**: 当前处于开发模式，可能存在性能优化和热更新问题

## 🐛 常见问题

### 无法访问界面
1. 检查服务是否正在运行: `ps aux | grep vite`
2. 检查端口是否监听: `netstat -tlnp | grep 5173`
3. 尝试使用其他 IP 地址访问

### 页面空白或加载失败
1. 打开浏览器开发者工具（F12）查看控制台错误
2. 检查 Mock API 服务是否运行: `ps aux | grep mock_api_server`
3. 清除浏览器缓存并刷新页面

### 登录失败
1. 确认使用正确的测试账号
2. 检查 Mock API 服务是否正常运行
3. 查看浏览器网络请求日志
